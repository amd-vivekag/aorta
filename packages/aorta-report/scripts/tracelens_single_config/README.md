# TraceLens Single Configuration Scripts

Shell scripts for orchestrating single-configuration trace analysis runs.

For the CLI reference (`aorta-report pipeline summary`), see the [User Guide](../../docs/user-guide.md#45-pipeline-commands).

## Scripts

| Script | Purpose |
|--------|---------|
| `compare_all_runs.py` | N-way comparison across multiple TraceLens runs (the CLI `pipeline summary` only supports 2-way baseline vs test) |
| `run_rccl_warp_speed_comparison.sh` | Shell orchestration for RCCL warp speed training sweep with multiple configurations |

### N-Way Comparison

The `compare_all_runs.py` script compares more than two runs at once, which is not yet supported by the `aorta-report` CLI:

```bash
python packages/aorta-report/scripts/tracelens_single_config/compare_all_runs.py \
    --inputs /path/to/run1 /path/to/run2 /path/to/run3 \
    --output /path/to/output
```

Each input directory must contain a `gpu_timeline_summary_mean.xlsx` (generated by `aorta-report process gpu-timeline`).
